Automatic Installation Scripts

WordPress

Execute this command to automatically install WordPress:

wget https://voutuk.github.io/bash/wp.sh && bash wp.sh

Zabbix

Execute this command to automatically install Zabbix:

wget https://voutuk.github.io/bash/zab.sh && bash zab.sh

Zabbix Docker

Download the Zabbix Docker installation script:

wget https://voutuk.github.io/bash/zab-docker.sh && bash zab-docker.sh
